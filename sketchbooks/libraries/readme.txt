For information on installing libraries, see: http://arduino.cc/en/Guide/Libraries

Easiest way with Arduino IDE 1.0.5:
1. Download library as zip file (or compress to zip after download)
2. In the Arduino IDE click "Sketch->Import Library->Add Library..."
3. Select the zip file that contains your library (ATTENTION: name of zip file   must be the same as your library's .cpp and .h file)
4. click on the "Ok" button
